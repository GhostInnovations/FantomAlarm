Fantom Alarm (Fire Alarm Navigational Tracking Outer Marker) is a solar-powered alarm system that detects fires in remote, rural or urban areas using infra-red technology. Within 60 seconds of fire ignition, the system relays real-time data such as exact location, temperature, speed and spread to a mobile phone application. The data is broken down and fed into an algorithm adding terrain type and windspeed/direction. The data is then translated and graphics are made available through the app in real-time allowing the system to accurately predict and project speed/spread and trajectory over set periods of time. This gives fire authorities a new and innovative tool to not only detect fires quickly but to control outbreaks and avert potential loss of life whilst reducing pollution, damage to habitats and property.

How The F.A.N.T.O.M. Works

The Base Node (Trunk) contains fold out solar cells powering a lithium battery, Infra-red semi sphere, GPS tracker and Bluetooth 5 receptor allowing leaves to be synced within a range of 1.5 kilometres and relays data to the App.

Trunk

The (Leaf/ Leaf Node) is a substantially smaller device that contains a solar cell strip powering a lithium battery, Infra-red semi sphere and Bluetooth 5 receptor allowing the device to be synced to the Trunk and or another Leaf. each system requires one Trunk and you can add as many leaves to cover an area of any size all relaying data to the Trunk.

Leaf

Each system requires one Base Node (Trunk) and any number of Leaf nodes (leaves) can then be synced within a range of 1.5km from each other creating a net system that can cover an area as large or small as you need, however the more leaves synced the more effective the system will work in not only early detection but gathering data which is then relayed to the Fantom App.
